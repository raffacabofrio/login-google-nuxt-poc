# LOGIN GOOGLE POC 


## Primeiros passos

```bash
# npm
npm install

# cli do nuxt
npm install -g nuxi

# roda local
npm run dev
```

## Production

Build the application for production:

```bash
# build estático + deploy no surge ( ver package.json para mais detalhes )
npm run deploy
```



## validação online
